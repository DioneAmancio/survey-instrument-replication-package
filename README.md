# survey-instrument-replication-package

# Replication Package

This repository contains the materials and supporting documents associated with the survey study reported in the submitted manuscript.

## Repository Contents

* **survey_raw_responses_and_coding.xlsx**: Contains the survey questions and the corresponding sanitized participant responses, with sensitive information removed to preserve participant anonymity.
* **codebook_category_analysis.xlsx**: Contains the codebook, categorization structure, and classification analysis used to organize and interpret the qualitative findings reported in the paper.

## Language Notice

The survey was conducted in Brazil with Brazilian software developers.
Therefore, the original survey instrument and related documentation are provided in **Portuguese**, reflecting the language used during data collection.

## Purpose

This repository is made available to support the peer review process and improve transparency and reproducibility of the research.
